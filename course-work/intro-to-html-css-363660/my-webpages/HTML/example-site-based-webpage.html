Now all we need is a submit button. You can even change the text on the button to say something.

HTML can create these forms, but cannot submit the information. You need PHP or mySQL or some backend to accomplish this. The submit button will allow you to add character codes in the url, andthe information entered to the form will be appended to the url when the submit button is pressed. It's interesting to note, but it does not save the information at all. It saves the information from the "named" elements of the form. So the element with the name 'm' is stored. Each item will need a separate name to store all the info in the url. Even for grouped elements, it's best to keep the names separate so that you can submit the correct selection.
